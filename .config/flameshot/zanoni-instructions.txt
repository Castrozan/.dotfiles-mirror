

For desktop environment that holds ownership of the print screen key, 
you can change the shortcut key to something else. 

For example, in GNOME, you can change the shortcut key by going to 
Settings > Keyboard Shortcuts > Screenshots. 

You can change the shortcut key to something else like Ctrl + Print Screen.

And then you can add a custom shortcut key for Flameshot. Like this:

Name: Flameshot
Command: flameshot gui
Shortcut: Print Screen
